# Canva Parity Specification

The Canva website design is the visual source of truth for the init / JPV-OS Access Gateway homepage.

Implementation standard:
- Do not approximate with generic SaaS sections.
- Do not use the old left-aligned hero.
- Do not let legacy `.home-product-page` styling drive the homepage.
- Build the homepage inside `.canva-parity-page`.
- Match the Canva layout: centered hero, compact metrics row, tight operational cards, cinematic grid/glow, dark enterprise shell.
- Keep "Welcome to the Venture." as the hero headline.
- Keep "Get init" as the primary CTA.
- Use Stripe/Discord language where operationally relevant.
